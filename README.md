# Private Memos v0.2.2

Private Memos 是一个桌面端私人备忘、任务、日程、剪贴板与记账管理工具。  
当前版本为 `v0.2.2`，支持 Windows x64 桌面端，并加入 Supabase 自动同步能力。

## 主要功能

- 私人笔记管理
- 任务管理与置顶任务
- 日程表与提醒
- 剪贴板模块
- 记账与本地账本
- 重点内容模块
- 本地密码保护
- Supabase 云端自动同步
- 桌面托盘运行
- 全局快捷键唤起剪贴板模块，默认 `Ctrl + E`
- 单实例运行，避免多开导致同步或本地数据冲突

## 版本信息

- 当前版本：`0.2.2`
- 平台：Windows x64
- 同步能力：桌面端 Supabase 自动同步版本
